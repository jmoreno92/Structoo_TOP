About Structoo

Structoo is a personal project to provide simple, transparent calculation helpers for structural and civil engineers. The goal is modest: clear inputs, documented logic where applicable, and outputs you can trace or paste into submittals and calculation packages.

Steel shape data (AISC)

The AISC shape database filter uses tabular properties derived from the AISC Shapes Database v16.0 (Excel workbook). AISC publishes this database as a reference to the Steel Construction Manual. Structoo is not affiliated with, endorsed by, or sponsored by AISC. Always verify section properties against the current Manual, official AISC publications, or your project specifications before relying on them for design.

If you use the AISC workbook yourself, comply with AISC’s license and terms for that file. Regenerate the JSON bundled with this site only from a copy you are permitted to use.
